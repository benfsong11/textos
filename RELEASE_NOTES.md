# Textos v0.2.0 릴리즈 노트

## 소개

Textos는 깔끔하고 집중력 있는 글쓰기 환경을 제공하는 미니멀 텍스트 및 마크다운 에디터입니다.

이번 v0.2.0은 WPF 기반에서 Electron으로 전환한 첫 번째 정식 릴리즈로, 전면적인 UI 재설계와 다양한 신규 기능이 포함되어 있습니다.

## 주요 기능

### 편집 모드
- **일반 텍스트 (.txt)**: 편집 모드와 페이지 뷰 모드 지원
- **마크다운 (.md)**: 실시간 미리보기 전환 지원

### 페이지 뷰
- A4 기반 다중 페이지 편집
- 자동 페이지 나눔 및 페이지 번호 표시
- 페이지 간 화살표 키 이동

### 파일 관리
- 새 문서 생성 시 파일 형식 선택 (.txt / .md)
- 열기, 저장, 다른 이름으로 저장
- 최근 파일 빠른 열기
- 저장하지 않은 변경사항 경고

### 설정
- 테마: 라이트 / 다크 / 시스템 설정 따름
- 글꼴 및 글꼴 크기 변경 (8pt ~ 72pt)
- 기본 보기 모드 설정
- 자간 및 줄간격 조절
- 글자 수 표시 방식 선택 (공백 포함 / 제외)

### UI
- 퍼플-블루 테마 기반 디자인
- 상태 바에 글자 수 표시
- 텍스트 정렬 (좌측 / 중앙 / 우측)
- Ctrl+마우스 휠 줌 지원
- 한국어 UI

### 크로스 플랫폼
- Windows, macOS 지원
- macOS 신호등 버튼 위치 대응

## 시스템 요구사항

- Windows 10 이상 (64bit)
- macOS 10.15 이상

## 설치

`Textos Setup 0.2.0.exe`를 다운로드하여 실행하세요. 설치 경로를 직접 지정할 수 있습니다.

## 알려진 사항

- 코드 서명이 적용되지 않았으므로 Windows SmartScreen 경고가 표시될 수 있습니다. "추가 정보" → "실행"을 클릭하여 진행하세요.
